Generador de Excusas

Un generador de excusas aleatorias es una herramienta que genera excusas de manera aleatoria para evitar una situación incómoda. Se puede usar en forma de proyecto en JavaScript o como aplicación.

    Cómo funciona

  • Genera aleatoriamente cada parte de la oración
  • Concatena los elementos de los arreglos en el orden correcto para crear una excusa consistente

    Usos

  • Para salir de un problema en una situación difícil
  • Para faltar a una reunión social o llegar tarde a una reunión
